Thinkful Projects

As a student with Thinkful, I will be completing one new portfolio project a week. The above "About Me" is my first project from the course. The following is what I have completed, in chronological order.

Hack the New York Times, Week 2

By "hacking" the NY Times using Developer Tools, I learned how to make live-edits to websites for effective debugging by inspecting elements.

Karma Landing Page, Week 3

The best way to learn is by copying. From cloning a portion of a landing page, I learned about semantic HTML and better ways to style.

Responsive Design and Portfolio, Week 4

This portfolio project serves as an excellent moment to define myself as a developer and nicely concludes the HTML/CSS portion of the course.

jQuery Street Fighter, Week 5

As an introduction to jQuery, I have created a fun Street Fighter landing page with a few animations and events.

Grocery List App, Week 6

Using jQuery, I made this grocery list app that will add, check off, and delete items. Fully-responsive.

FizzBuzz, Week 7

As an introduction to Javascript, I wrote out the logic for a game called "FizzBuzz". In this game, the user is prompted for a number. The browser will then spit out "Fizz" if the number is divisible by 3, "Buzz" if the number is divisible by 5, "FizzBuzz" if divisible by both, or simply the number. The browser will instruct users to refresh the page and try again if the number is invalid.

Hot or Cold, Week 8

Hot or Cold is a game where the user has to guess the number. The game will provide feedback as the user continues to make guesses. Thinkful provided all of the boiler plate for this project, so I was able to focus solely on creating functions and conditionals. This game was created using jQuery. If you check the console, you will see my process for debugging and problem solving.

Quiz App, Week 9

The purpose of this exercise was the exploration of click events, arrays and conditional logic. I made a quiz about flowers because I like them :)
